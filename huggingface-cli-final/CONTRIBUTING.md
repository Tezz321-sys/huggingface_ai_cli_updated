# Contributing to AI Free CLI

First off, thank you for considering contributing to this tool! Whether you are a python expert or just improving the documentation, it is people like you that make open-source a great community.

## How Can I Contribute?

### 1. Reporting Bugs
If you find a new bug (and it isn't listed in our `TROUBLESHOOTING.md`), please create a new **Issue** on GitHub! Please include:
- Your Operating System (Windows, Kali Linux, macOS)
- The exact error code or behavior
- Steps to reproduce the freeze or crash

### 2. Suggesting Enhancements
Have an idea for a cool new feature? Open an Issue and tag it as an enhancement to discuss it! Some ideas to work on:
- Adding Text-To-Speech (TTS) integration
- Image Generation commands
- Local saving of chat logs to text files

### 3. Submitting Pull Requests (Code changes)
If you want to write the code yourself:
1. **Fork** the repository on GitHub.
2. **Clone** it to your local machine.
3. Create a new branch for your feature (`git checkout -b feature/CoolNewFeature`).
4. Commit your changes (`git commit -m 'Added this awesome feature'`).
5. Push to the branch (`git push origin feature/CoolNewFeature`).
6. Open a **Pull Request** on this repository!

## Setting up for Development
To modify the code directly on your local machine:
1. Edit the core engine file directly at `ai_cli/main.py`.
2. After making your edit, run `pip install -e .` to push the changes onto your terminal executable!
3. Test heavily before pushing a Pull Request.
